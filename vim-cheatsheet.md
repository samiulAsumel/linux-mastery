# Vim Cheatsheet - Quick Reference Guide

## Modes

- Normal: default (navigate + commands)
- Insert: i (type text)
- Command: :(save, quit, search)
- Visual: v (select text)

## Essential Commands

:wq = save + quit
:q! = quit without save
i = insert mode
Esc = back to normal mode
dd = delete line
yy = copy line
p = paste after cursor
u = undo
gg = first line
G = last line
/word = search
:%s/old/new/g = replace all
