# Linux Filesystem Hierarchy - Mongla Port Reference

## Critical Directories

| Directory | Purpose            | Port Analogy          |
| --------- | ------------------ | --------------------- |
| /etc      | Config files       | Operation Manuals     |
| /var/log  | System logs        | Activity Logs         |
| /home     | User dirs          | Staff Quarters        |
| /tmp      | Temp Files         | Temp holding area     |
| /root     | Root home          | Director's Office     |
| /proc     | Live kernel info   | Live status board     |
| /boot     | Kernel + GRUB      | Control Tower         |
| /dev      | Device files       | Equipment Registry    |
| /usr      | User programs      | Tool Shed             |
| /bin      | Essential binaries | Core Tools            |
| /sbin     | System binaries    | Maintenance Tools     |
| /lib      | Shared libraries   | Shared Resources      |
| /opt      | Optional software  | Specialized Equipment |
| /media    | Removable media    | Docking Station       |
| /mnt      | Mount points       | Temporary Storage     |
